gen_dfu_binary_for_win.exe(gen_dfu_binary_for_linux)

This utility converts spl/scpu/ncpu/model binary to DFU format
windows10   : gen_dfu_binary_for_win.exe
Ubuntu 16.04: gen_dfu_binary_for_linux

Usage 0: gen_dfu_binary_for_win.exe -help      # for usage 1-4
usage 1: gen_dfu_binary_for_win.exe -scpu scpu_in_file scpu_out_file
usage 2: gen_dfu_binary_for_win.exe -ncpu ncpu_in_file ncpu_out_file
usage 3: gen_dfu_binary_for_win.exe -model fw_info_file_in  all_model_file_in model_out_file
usage 4: gen_dfu_binary_for_win.exe -spl spl_file_in  spl_out_file
